컴파일 실행 후 콘솔(Console) 창 출력 메시지 의미
Process finished with exit code 0
직역하면 종료 코드 0으로 프로그램이 종료되었다는 것인데,
보통 0 은 에러 없이 프로그램이 성공적으로 실행되었다는 것을 의미함.
그래서 위 메시지가 나오면 프로그램이 잘 실행되었다고 생각하면 된다.
Process finished with exit code 1
직역하면 종료 코드 1로 프로그램이 종료되었다는 것인데,
보통 1 은 에러 발생하여 프로그램이 정상적으로 실행되지 못하는 것을 의미함.
그래서 위 메시지가 나오면 프로그램이 오류가 발생했다고 생각하면 된다.

백슬래시(\) 출력 예시
\\과 같이 쓰는건 escape character 라고 해서 \ 와 어떤 문자를 결합하여 문자열내에서 다른 의미로 쓰이게 해준다.
문자열로 출력해야 하는 글자가 역슬래시, 작은따옴표, 큰따옴표 이렇게 세 개지만
문자열을 작은따옴표(')로 감싸면 큰따옴표(")는 저절로 문자열로 출력되기 때문에 아래와 같이 역슬래시와 작은따옴표에만 역슬래시를 붙이시면 된다.
